Gameduino image encoder

This utility encodes an image as a Gameduino character map. A summary of what is valid:

  • width and height must be multiple of 8
  • each 8x8 character can use at most 4 colors (although this utility will palettize each 8x8 cell if necessary)
  • no more than 256 unique characters.
See Gameduino for Artists for more details
